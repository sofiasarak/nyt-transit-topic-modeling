# New York City Transportation Topic Modeling

Here, I use an LDA model to explore changes in New York Times reporting, as it relates to tranportation, over time. 

Data was sourced from the NYT API. More information can be found in the `topic_modeling.Rmd`!

`word_frequency.png` is one of the results from this analysis, and shows the frequency of the top 10 words in each topic from 2020 compared to 2026.

This project was completed as part of the final for MEDS course [EDS231: Text and Sentiment Analysis for Environmental Problems](https://maro406.github.io/EDS-231-text-sentiment/index.html) with lecturer Mateo Robbins.
